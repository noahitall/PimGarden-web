Software Vendors and Service Providers

Introduction

PimGarden utilizes various third-party software vendors and service providers to deliver our application and services. This document provides transparency about the vendors we work with and rely on.

Vendor List

Domain and DNS

Namecheap - Used for domain registration and DNS management services.

Hosting

Vercel - Cloud platform used for hosting our web application and services.

App Distribution

Apple App Store - Distribution platform for our iOS application.

Google Play Store - Distribution platform for our Android application.

Development and Build Tools

Expo - Platform for React Native application development, building, and distribution.

Cursor/Claude - AI-assisted development tools used to assist in coding and development processes.

Code Repository

GitHub - Platform used for code hosting, version control, and collaboration.

Vendor Updates

This list may be updated from time to time as we add or remove vendor relationships. Please check back periodically for the most current information.

Vendor Privacy Practices

Each vendor maintains their own privacy policies and terms of service. While we make efforts to work with reputable vendors who maintain strong privacy and security practices, we encourage users to review the privacy policies of these third parties directly.

Questions

If you have questions about our vendors or how we use their services, please contact us at privacy@pimgarden.com.
